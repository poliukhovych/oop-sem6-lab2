# oop-sem6-lab2
This project is a **full-stack web application** built with next technologies:
### Backend
- **ASP.NET Core** – Web API with MVC-style Controllers
- **Entity Framework Core** – ORM for database access
- **PostgreSQL** – Relational database
- **ASP.NET Middleware** – For authorization, logging and error handling
- **Auth0** – Authentication and role-based access (admin, subscriber)
- **xUnit** – Unit testing framework for .NET

### Frontend
- **React** – User interface library
- **Vite** – Build tool and development server
- **Bootstrap** – Component-based CSS framework
- **JavaScript** – For all frontend logic

### Dev Environment
- **Docker + Dev Containers** – Backend, frontend, and database are containerized for easier development
- **Visual Studio Code** – Primary development environment
